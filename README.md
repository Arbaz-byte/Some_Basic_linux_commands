# Some_Basic_linux_commands
Summarizing the most frequently used linux commands for DevOps Engineers. It conatins syntax and their describtion and use case scenario.

1. **ls(List)** : To view files and folder within the directory.<br>
   Common flags:<br>
    i. **ls -l**: Long formate, shows detailed list of file permissions,owner name,file size(in Bytes), last modification.<br>

    ii. **ls -a**: Shows everything, including hidden files (starts with .git or .bashrc)<br>

    iii. **ls -lth** : shows detailed info, with human readable files sizes, shorted by the most recently modified time.
